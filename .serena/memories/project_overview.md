# Lattice_release overview
- Purpose: repository snapshot for Lattice marketing/support materials, primarily README/changelog content and GitHub Pages site under `docs/`.
- Product: Lattice is a native Swift macOS reference manager focused on lightweight literature management, metadata enrichment, annotation extraction, export templates, and Apple Shortcuts workflows.
- Current repo structure is content-heavy rather than app-source-heavy: `README.md`, `README.zh-CN.md`, changelogs, `lattice_intro/` screenshots, and `docs/` static site assets.
- Website hosting model: GitHub Pages from `docs/`, plain static HTML/CSS/JS with no build step.
- Reusable assets: screenshots and Mac App Store badge in `lattice_intro/`, copied website-safe assets in `docs/assets/`.
