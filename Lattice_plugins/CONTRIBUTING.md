## What You Can Contribute

- **New plugins** built on the Lattice Scholar local API (any platform or language)
- **Improvements** to the existing Word add-in
- **Bug fixes** and performance improvements
- **Documentation** and examples

## Submitting Changes

1. Fork the repository.
2. Create a branch for your changes.
3. Submit a pull request with a clear description of what your change does and why.

## Third-Party Dependencies

If your contribution introduces new third-party dependencies, ensure they are compatible with the project's licensing and document them in [THIRD_PARTY_NOTICES.md](./THIRD_PARTY_NOTICES.md). If a new dependency requires user-visible attribution (for example CPAL §14, GPL family, or similar), update the Credits panel in the Word add-in (`word-addin/web/taskpane/taskpane.html`) accordingly.
