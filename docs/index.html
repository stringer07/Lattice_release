Technical support page for the Lattice Scholar macOS app.

Lattice Scholar is a lightweight literature manager built for macOS.

Support contact: stringer_fan@qq.com

Project page: GitHub Repository

Frequently Asked Questions

1. Which macOS versions are supported?

Lattice Scholar is designed for macOS 15.0 or later.

2. How do I report a bug or request a feature?

Please send an email to stringer_fan@qq.com with your macOS version, app version, and a short description of the issue.

3. What information should I include in a support request?

Please include:

  • Your macOS version
  • The app version
  • Steps to reproduce the problem
  • A screenshot if relevant

4. Where can I learn more about the app?

You can visit the GitHub repository for release notes and project details.
